# Portfolio Website

This is my personal portfolio website written entirely in Markdown, built using the [Material for MkDocs](https://github.com/squidfunk/mkdocs-material) framework. It showcases my projects, skills, and background in a clean, professional static site.

## Features

- Content written in simple Markdown for easy editing
- Responsive design that works on desktop, tablet, and mobile devices
- Powered by the Material for MkDocs theme for a modern and accessible look
- Fast and lightweight site generation with excellent performance
- Fully customizable through simple configuration

## Deployment

The site is deployed securely and efficiently using [Cloudflare Pages](https://pages.cloudflare.com/), ensuring fast global delivery and HTTPS by default.
